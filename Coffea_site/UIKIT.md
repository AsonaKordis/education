# UI Kit (User Interface Kit)

UI Kit (User Interface Kit) - это набор готовых элементов интерфейса, который дизайнер использует при создании сайта или приложения.

## Что входит в UI Kit

### Обычно UI Kit содержит:

* Цветовая палитра
* Главные цвета (Primary)
* Дополнительные (Secondary)
* Фоновые
* Цвета текста
* Статусы (успех, ошибка, предупреждение)

### Типографика:

* Виды шрифтов (например, Inter, Roboto)
* Размеры текста (H1, H2, H3, body)
* Межстрочные интервалы
* Жирность текста

### Компоненты интерфейса

Это самые важные элементы, которые дизайнер использует постоянно:

* Кнопки (primary, secondary, outline)
* Поля ввода (input, textarea)
* Переключатели (switch, checkbox, radio)
* Выпадающие списки (dropdown)
* Навигация (меню, хедер)
* Карточки товаров / статей
* Формы
* Модальные окна

### Сетка и отступы

* правила, какие расстояния использует дизайн (например, 8px grid)
* размеры контейнеров
* колоночная сетка (12 колонок, 24, и т.д.)

### Стилевые правила

* Скругления (4px, 8px, 16px)
* Тени элементов
* Использование цвета
* Примеры правильного и неправильного использования элементов

## Зачем нужен UI Kit?

1. Ускоряет работу

Ты не рисуешь кнопку заново — просто берёшь готовую.

2. Держит стиль единым

Все страницы сайта выглядят одинаково и профессионально.

3. Упрощает командную работу

Разные дизайнеры используют одни и те же элементы — нет хаоса.

4. Легко расширять проект

Нужно добавить новый элемент? Он автоматически вписывается в стиль.

## Где используется UI Kit?

* В Figma
* В Webflow
* В React-компонентах
* В мобильных приложениях
* В больших дизайн-системах (Material UI, Ant Design, Fluent)
