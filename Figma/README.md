# Что такое RGB в целом

RGB — это модель цвета, основанная на смешивании трёх цветов света:

* R — Red (красный)

* G — Green (зелёный)

* B — Blue (синий)




Когда эти цвета смешиваются в разных пропорциях, получается любой цвет, который мы видим на экране.
Например:

* RGB(255, 0, 0) — чисто красный

* RGB(0, 255, 0) — чисто зелёный

* RGB(0, 0, 255) — чисто синий

* RGB(0, 0, 0) — чёрный

* RGB(255, 255, 255) — белый


Когда мы выбираем цвет любого объекта (текст, форма, фон), справа в панели свойств есть блок Fill (Заливка). Там можно менять формат отображения цвета:

* HEX — шестнадцатеричный формат, например #FF5733

* RGB — три числа от 0 до 255

* HSL — оттенок, насыщенность, яркость


Чтобы включить именно RGB:

![Figma Colors 02.png](../Photo/Figma%20Colors%2002.png)

* Выдели объект.

* В правой панели найдите Fill.

* Нажмите на цветовой квадратик.

* В выпадающем меню формата выберите RGB.


Теперь вам покажут три поля: R, G, B, куда можно вводить значения.


Как использовать RGB новичку:
1. Подбирать оттенки вручную

Если вы двигаете ползунок цвета — значения R, G, B меняются автоматически.
Можно экспериментировать и смотреть, как изменение одного компонента влияет на цвет.

2. Точно вводить цвет по заданию

Например, у клиента цвет бренда — RGB(34, 136, 204).
Вы просто вводите эти три числа — и получаете точное совпадение.

3. Работать с прозрачностью

В Figma рядом с RGB есть поле A — прозрачность (Alpha).

100% — полностью непрозрачный

0% — полностью прозрачный

Например: RGBA(255, 0, 0, 20%) — красный на 20% прозрачности.
